🚀 HOW TO USE:

1. Restart IDM to load the new code

2. Add a download using "Add URL" (Ctrl+N)

3. Enter URL and click "Start Download"

4. FileInfoDialog appears showing:
   • Filename
   • File size
   • Content type
   • Resume support status
   • Save location
   • Category selector

5. Review the information and choose:
   • "Start Download" - Begin now
   • "Download Later" - Queue it
   • "Refresh" - Get latest info
   • "Cancel" - Go back

6. To refresh any existing download:
   • Right-click on it in the downloads list
   • Select "Refresh File Info"
   • Dialog shows current server info
